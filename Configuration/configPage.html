                        Jellyseerr (optional)
                        

                            Preferred request method — routes to Radarr/Sonarr internally. When configured, ❤️ favouriting a recommendation stub also submits a Jellyseerr request on the next sync.
                        

                        

                            
                            
e.g. http://192.168.1.100:5055 or https://requests.yourdomain.com

                        

                        

                            
                            
Jellyseerr → your profile (top-right) → API Key tab

                        

                        
                        

                    


                    
                    

                        Radarr (optional, direct)
                        

                            Only configure if you are not using Jellyseerr. When Jellyseerr is configured it handles movie requests internally.
                        

                        

                            
                            
e.g. http://192.168.1.100:7878

                        

                        

                            
                            
Settings → General → Security → API Key

                        

                        

                            

                                
                                
                            

                            
                        

                        

                            
                            
e.g. /movies — must match a root folder in Radarr. 

                        

                        
                        

                    


                    
                    

                        Sonarr (optional, direct)
                        

                            Only configure if you are not using Jellyseerr. When Jellyseerr is configured it handles TV show requests internally.
                        

                        

                            
                            
e.g. http://192.168.1.100:8989

                        

                        

                            
                            
Settings → General → Security → API Key

                        

                        

                            

                                
                                
                            

                            
                        

                        

                            
                            
e.g. /tv — must match a root folder in Sonarr.
